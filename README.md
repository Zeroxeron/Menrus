### Описание [RU]
Menrus распознает сообщения чата на русском языке, которые предположительно являются командами, и **выполняет** их.
> ".рщьу" -> "/home"
> 
> "/рудз" -> "/help"
> 
> ".spawn" -> "/spawn"

### Права и команды
`menrus.admin` 
> /menrus reload
> 
> /menrus version 

*Настроить сообщения можно в конфиг-файле* **config.yml**

### Description [EN]
Menrus recognizes occasional chat messages in russian that suppose to be a command and therefore **executes** them.
> ".рщьу" -> "/home"
> 
> "/рудз" -> "/help"
> 
> ".spawn" -> "/spawn"

### Permissions & commands
`menrus.admin` 
> /menrus reload
> 
> /menrus version 

*You can setup plugin messages in* **config.yml**
